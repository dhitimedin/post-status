== Installation ==

1. Upload the entire `post-status` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Settings** menu under Tools menu in your WordPress admin screen.
